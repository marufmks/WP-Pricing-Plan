Create and manage beautiful pricing plans with an easy-to-use interface. Perfect for showcasing your products and services pricing.

== Description ==

Pricing Plan is a modern WordPress plugin that allows you to create and manage beautiful pricing plans for your products and services. With its intuitive interface and powerful features, you can easily create, customize, and display pricing tables on your website.

= Features =

* Modern, card-based admin interface
* Fully responsive design
* Multiple pricing plan layouts
* Package management system
* Easy-to-use shortcodes
* Fast and lightweight
* Developer-friendly

= Pro Features =

* Additional pricing table templates
* Advanced styling options
* Custom CSS support
* Priority support
* Regular updates

= Usage =

1. After activation, find "Pricing Plan" in your WordPress admin menu
2. Create a new pricing plan
3. Add packages to your plan
4. Use the shortcode `[pricing_plan id="X"]` to display your plan
5. Customize appearance using the settings page

= Available Shortcodes =

* `[pricing_plan id="1"]` - Display a specific pricing plan
* `[pricing_plan id="1" type="table"]` - Display as comparison table

= For Developers =

The plugin includes a REST API for custom integrations:

* List all plans: `GET /wp-json/pricing-plan/v1/plans`
* Create a plan: `POST /wp-json/pricing-plan/v1/plans`
* Get a specific plan: `GET /wp-json/pricing-plan/v1/plans/{id}`
* Update a plan: `PUT /wp-json/pricing-plan/v1/plans/{id}`
* Delete a plan: `DELETE /wp-json/pricing-plan/v1/plans/{id}`

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/pricing-plan` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Pricing Plan menu item to configure the plugin
4. Use shortcodes to display pricing plans in your posts/pages

== Frequently Asked Questions ==

= Can I create multiple pricing plans? =

Yes, you can create as many pricing plans as you need.

= Is it responsive? =

Yes, all pricing plans are fully responsive and look great on all devices.

= Can I customize the colors and styles? =

Basic styling options are available in the free version. For advanced customization options, consider upgrading to Pro.

= Does it work with my theme? =

Yes, the plugin is designed to work with any properly coded WordPress theme.
